# 🧋 奶茶记录册 - 产品需求文档

## 项目概述

一款运行在 macOS 上的奶茶记录桌面应用。用户可以通过日历形式记录每天喝过的奶茶，支持评分、评论、图片、搜索、置顶等功能。数据绑定到用户账号，支持跨设备迁移。

## 目标用户

- 喜欢喝奶茶、想记录消费习惯的个人用户
- 非技术背景，需要直观友好的操作体验

## 功能需求

### F1. 账号系统
- 邮箱 + 密码注册/登录
- 登录状态持久化（关闭 App 后无需重新登录）
- 登出功能

### F2. 添加奶茶记录
- 输入奶茶名称（必填）
- 添加图片：支持拖拽文件 + 剪贴板粘贴
- 选择日期（默认当天）
- 星级评分（1-5 颗星，可选）
- 文字评论（可选）
- 图片上传前自动压缩

### F3. 日历视图
- 月视图日历，作为主界面
- 有记录的日期显示绿色圆点标记
- 点击日期查看当天记录
- 月份前后翻页
- 点击"今天"快速回到当前日期

### F4. 奶茶列表
- 按时间倒序排列（最新的在最上面）
- 每条记录显示：图片缩略图、名称、星级、评论、操作按钮
- 点击图片可放大查看

### F5. 时间范围筛选
- 3天
- 一周
- 两周（默认）
- 一个月

### F6. 置顶功能
- 可置顶最喜欢的奶茶
- 置顶项显示在侧边栏
- 可取消置顶

### F7. 搜索功能
- 按名称搜索
- 模糊匹配
- 搜索结果按时间倒序

### F8. 删除功能
- 删除某条奶茶记录
- 删除前弹出确认对话框
- 同时删除关联的图片

### F9. 编辑功能（未来）
- 编辑已有记录的名称、评分、评论、日期

### F10. 数据导出（未来）
- 导出为 PDF / CSV

## 非功能需求

### NF1. 性能
- 应用启动时间 < 3 秒
- 界面操作响应 < 200ms
- 图片加载有缩略图优化

### NF2. 安全性
- 用户只能访问自己的数据（Supabase RLS）
- 密码安全存储（Supabase Auth 处理）

### NF3. 可用性
- 空状态友好提示
- 错误状态友好提示
- 加载状态有动画反馈
- 支持键盘快捷键（Cmd+N 新建，Esc 关闭弹窗）

### NF4. 视觉风格
- 主色调：淡绿色 (#66BB6A)
- 风格：活泼、温暖、圆润
- 字体：系统默认中文字体
- 动画：微交互、过渡动画
